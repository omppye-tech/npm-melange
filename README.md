# Melange

A wrapper to install [melange-re/melange](https://github.com/melange-re/melange) from npm

## Usage

I created this installer for personal use, but if it is of any use to you, run:

### Global

```sh
$ npm i -g npm-melange
$ bsb
```

### Local

```sh
$ npm i -D npm-melange
$ npx bsb
```
